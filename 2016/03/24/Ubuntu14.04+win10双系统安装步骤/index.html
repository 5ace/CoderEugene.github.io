Ubuntu14.04+win10双系统安装步骤

发表于 2016-03-24   |   分类于 Ubuntu   |  


由于最近在学习操作系统会用到linux并且我本身也感兴趣，所以就安装了ubuntu,在这里我也写下我的安装教程（包括了双系统和虚拟机），也算是个人总结，希望能有些帮助！

下载Ubuntu

选择14的版本其实是为了稳定一些，不过在安装过程中还是出了许多问题。。

  • Ubuntu 14.04.4

使用UltralSO制作u盘镜像

  • 打开UltralSO （注意使用管理员模式打开）
  • 选择文件-》打开
  • 选择你要安装的U盘-》选择启动-》写入硬盘
  • 选择写入方式和便捷写入
  • 点击写入即可

分配磁盘空间

  • 右键新加卷-》压缩卷-》填入分配的空间（我分配了30G左右）

安装镜像

  • 电脑插入刚才制作好的U盘，然后重新启动电脑，按F12进入Windows boot manager,选择USB启动
  • 选择中文简体语言，连接wifi，然后选择 ‘其他选项’(这里如果你不会磁盘分区，也可以直接选第一项，系统会自动分区)
  • 选择刚才分配出来的空闲空间，点击左下角的+号3次分区（每次都是空闲中分区）
    • 第一次分区：点击+号，选择交换空间，大小我给了2048MB
    • 第二次分区：点击+号，选择/boot，大小我给了200MB
    • 第三次分区：点击+号，选择/，大小就将剩余空间都给了吧
  • 对于下面的安装启动引导器，我是默认的，大概是ubuntu来引导window,如果是这样的话，在卸载ubuntu时需要清理引导才能格式化ubuntu，但是我后来卸载了也不知道有没有更改引导引导，但是我格式化了ubuntu还是没什么问题，然后用EasyBCD将启动时的ubuntu给删了就行了，但是大家最好还是多了解这方面（安装完后我是设置了将windows作为第一启动项）
  • 接下来也就是自己设置一些语言什么的了

结语

  • 到了这里，ubuntu也就安装完了，如果顺利的话，你应该能用ubuntu了，但是，由于各种原因，我还是卸载了ubuntu,后来安装在了虚拟机中。

遇到的一些问题

没错，我就是遇到了这么多问题，所以最后卸载了它，并且装在了虚拟机当中，因为虽然解决了这些问题，但是我在运行时经常卡死。。。

  • 启动时卡死，出现Ubuntu is running in low-graphics mode
    • 这应该是显卡驱动的原因，我的解决办法是多试几次进入图形界面后更新驱动
  • 启动时黑屏
    • 这大概是高版本内核问题，我的解决办法是在进入ubuntu时选择高级模式，选了一个较低版本就可以了
  • 关机时卡死

附加-虚拟机中安装ubuntu

所需软件和镜像
  • 虚拟机：VM12
  • ubuntu15.04
安装步骤

相对于双系统，在虚拟机中安装会方便很多

  • 打开VM，选择新建虚拟机

  • 选择典型安装

  • 选择镜像ubuntu15.04

  • 输入信息

  • 接下来就是漫长的安装了，虚拟机的安装真是挺方便和实用的，当你安装好后需要安装VMtool，安装步骤我就不写了，因为网上已经有教程了，我这里链接一个就行了,切换语言也在这里
    • Ubuntu下 正确安装VMware Tools
    • ubuntu切换中文语言
结语

我用起来虚拟机非常流畅，也没有出现任何问题，重要的是便于安装和卸载，所以如果不是特别的需求，直接装进虚拟机吧
