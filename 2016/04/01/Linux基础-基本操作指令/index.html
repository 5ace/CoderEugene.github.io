Linux基础-基本操作指令

发表于 2016-04-01   |   分类于 Ubuntu   |  


简单的讲述一些比较基础并且比较常用和重要的操作指令，尽快熟悉这些指令可以让我们更快的熟悉linux

使用文字登录界面

  • Linux预设的情况下会提供六个Terminal来让使用者登入， 切换的方式为使用：[Ctrl] + [Alt] + [F1]~[F6]的组合按钮，它们分别对应文字接口登入 tty1 ~ tty6 终端机（执行等级run level 3)
  • [Ctrl] + [Alt] + [F7] ：切换回图形接口桌面（执行等级run level 5）
  • 经过小伙伴的提醒，由于我下载的是有图形界面的，通过[Ctrl] + [Alt] + [T]来打开控制台会好一些，会有高亮等功能

创建root管理员

  • 首先先进入其中一个文字登录界面，登陆当前账号，输入当前用户密码
  • 在终端输入 [sudo passwd root]
  • 输入你的当前密码
  • 成功后输2次密码，改密码为你的root密码
  • 在终端输入 [su root] ，然后输入root密码，这时候就获得了root权限

控制台输出中文

[Ctrl] + [Alt] + [T]来打开控制台,输入 [LANG=zh_CN.utf-8] 就可以正常显示中文（可以输入cal日历指令来检测是否输出成功）
